# Notes

> Древовидное приложение для заметок. Заметки живут внутри других заметок, как в проводнике. Можно перетаскивать их, писать в блоках и притворяться, что это Notion.

**Демо:** [notes-nine-lac.vercel.app](https://notes-nine-lac.vercel.app/)

[English version](README.md)

## Что это

Это иерархическая штука для заметок. Слева: дерево, которое можно перетаскивать. Справа: блочный редактор, где можно писать абзацы и заголовки. Заметки вкладываются в заметки. Блоки разделяются, объединяются и автоматически сохраняются.

В общем, всё.

Можете пользоваться, но будете мазохистом. Запросы вне США медленные, интерфейс так себе, а половина функций, которые ожидаются от нормального приложения для заметок, ещё не реализована.

Оно существует, потому что я хотел научиться делать:

- Дерево, которое не падает при перетаскивании
- Оптимистичные обновления, которые спокойно откатываются, когда сервер говорит «нет»
- Блочный редактор с настоящими CRUD-операциями
- Реляционную модель данных, которая работает с вложенными иерархиями без хаков со строковыми путями

## Что тут крутое

- Closure table для иерархии заметок
- Drag & drop с оптимистичными обновлениями и откатом
- Блочный редактор с разделением, объединением, переупорядочиванием, автосохранением
- LexoRank для стабильного упорядочивания соседних элементов
- Result-based обработка ошибок
- Shoo авторизация

## Стек

Next.js 16, React 19, Tailwind CSS 4, TanStack Query, Zustand, dnd-kit, Drizzle ORM, PostgreSQL (Neon), Zod, Bun.

## Запуск локально

```bash
bun install
bun db:push
bun dev
```

Нужен `.env` с `DATABASE_URL` и `APP_ORIGIN`.

## Статус

Сначала архитектура, потом взаимодействие, функционал намеренно неполный. Поиска нет. Форматированного текста нет. Тестов нет. Но ядро модели и редактор работают надёжно.
